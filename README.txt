Добро пожаловать в Weather bot!

ОПИСАНИЕ -------------------------------------------------------------------------------------
Данный телеграмм бот поможет вам узнать погоду и информацию о городах.


УСТАНОВКА ------------------------------------------------------------------------------------
Для установки программы необходимо скачайте файлы с git_hub в одну папку, запустите файл telega.py.


ИСПОЛЬЗОВАНИЕ --------------------------------------------------------------------------------
Найдите в телеграмме нашего телеграмм бота(@TestWeatherBot1000_bot) и напишите /start. 
После этого бот начнет свою работу и отправит вам сообщения на которые надо ответить.
